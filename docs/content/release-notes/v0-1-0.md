## What's in it

`pypi` reads the Python Package Index through its open public API, HTML search,
and RSS feeds. No API key required.

### Commands

| command | what it does |
|---|---|
| `pypi search <query>` | Search PyPI (HTML scrape), ranked results |
| `pypi package <name>` | Show latest release metadata for a package |
| `pypi releases <name>` | List all releases, sorted newest-first |
| `pypi files <name> [version]` | List distribution files for a release |
| `pypi deps <name>` | List declared dependencies (requires_dist) |
| `pypi updates` | Recent package updates from RSS |
| `pypi newest` | Newest packages added to PyPI from RSS |

All commands support `-o table|json|jsonl|csv|tsv|url|raw`, `--fields`,
`--template`, `-n/--limit`.

## Install

```bash
go install github.com/tamnd/pypi-cli/cmd/pypi@v0.1.0
```

Or download a prebuilt binary from the [releases page](https://github.com/tamnd/pypi-cli/releases/tag/v0.1.0).
